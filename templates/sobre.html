Micro SO para Microcontroladores en MicroPython

Objetivo del Proyecto: El proyecto tiene como objetivo desarrollar un sistema operativo ligero (Micro SO) para microcontroladores como el ESP32, Raspberry Pi Pico, y otros similares, utilizando MicroPython como lenguaje de programación principal. Este Micro SO está orientado a funcionar como un sistema "kiosk", permitiendo la instalación y gestión de aplicaciones o programas a través de una API servidor.

Características Principales:

  • Lenguaje y Entorno:
    • MicroPython: El sistema está desarrollado en MicroPython, un lenguaje que adapta Python para funcionar en microcontroladores, lo que permite una programación más sencilla y eficiente en hardware de recursos limitados.
    • Microdot: Se utiliza Microdot, un microframework inspirado en Flask, para manejar las operaciones de red y servir la API que permitirá la instalación y gestión de las aplicaciones.
  • Compatibilidad con Hardware:
    • ESP32, Raspberry Pi Pico, y Otros Microcontroladores: El sistema está diseñado para ser compatible con una variedad de microcontroladores populares en proyectos de IoT, facilitando su implementación en diferentes dispositivos con recursos de hardware limitados.
  • Sistema Kiosk: El Micro SO actúa como un sistema kiosk, proporcionando una interfaz simplificada y controlada para los usuarios finales. El objetivo es ofrecer una experiencia de uso cerrada, donde las aplicaciones se ejecuten de manera controlada y sin acceso directo al sistema subyacente.
  • Gestión de Aplicaciones:
    • Instalación y Actualización vía API: Las aplicaciones y programas se pueden instalar y actualizar a través de una API servidor, lo que permite la gestión remota de las aplicaciones sin necesidad de intervención física en el dispositivo.
    • Repositorio Centralizado: Los programas y aplicaciones compatibles se almacenan en un servidor centralizado, desde donde se pueden descargar e instalar en los dispositivos que ejecutan el Micro SO.
  • Seguridad y Control:
    • Aislamiento de Aplicaciones: Cada aplicación se ejecuta en un entorno controlado para evitar que interfiera con el sistema o con otras aplicaciones, asegurando así la estabilidad y seguridad del Micro SO.
    • Actualizaciones Seguras: El sistema implementa un mecanismo para verificar la integridad de las aplicaciones antes de su instalación o actualización, garantizando que solo se ejecuten software seguros y aprobados.

Aplicaciones del Proyecto:

Este Micro SO es ideal para:

  • IoT y Dispositivos Inteligentes: Implementación en dispositivos IoT donde se requiere una gestión sencilla y segura de las aplicaciones.
  • Entornos de Exposición o Puntos de Venta: Utilización en kiosks, terminales de autoservicio, y otros entornos donde se necesita una interfaz limitada y controlada para el usuario final.

Futuras Expansiones:

  • Soporte para más Hardware: Ampliar la compatibilidad a otros microcontroladores y sistemas embebidos.
  • Interfaz de Usuario: Desarrollo de una interfaz gráfica ligera para facilitar la interacción del usuario con el sistema y las aplicaciones instaladas.
